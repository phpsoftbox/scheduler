# CLI

Команда планировщика:

```
schedule:run
```

Опции:
- `--time` (`-t`) — запуск «как будто сейчас» в заданное время (например, `2024-01-01 12:00:00`)
- `--timezone` (`-z`) — часовой пояс (например, `Europe/Moscow`)

Команда загружает расписания через `ScheduleLoader` и выполняет все задачи,
которые должны быть запущены в указанное время (по умолчанию — текущее).

`--time` полезен для ручного запуска, отладки и прогонов «задним числом»
без ожидания реального времени. В обычном cron-запуске опция не нужна.

Если в расписании используются `command()`, `schedule:run` автоматически
подставляет runner для выполнения CLI-команд.
