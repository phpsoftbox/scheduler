# About

`phpsoftbox/scheduler` — компонент для запуска задач по расписанию. Он не содержит циклов и таймеров, а предоставляет механизм проверки времени и запуска задач по переданному времени.

Основные элементы:
- `Scheduler` — регистрирует задания и запускает их по расписанию
- `ScheduledTask` — настройка расписания (every/daily/weekly/quarterly/yearly)
- `ScheduledGroup` — несколько задач с одним расписанием и таймзоной
- `CronExpression` — проверка cron-выражений
- `ScheduleLoader` — загрузка расписаний из файлов

Опционально:
- поддержка DI для invokable-классов обработчиков
- блокировки задач через `CacheInterface`
- фоновое выполнение через `Queue` (если установлен)
